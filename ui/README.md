# Quasar 框架 UI 包

> 在创纪录的时间内构建高性能 VueJS 用户界面：响应式单页应用程序、SSR 应用程序、PWA、浏览器扩展、混合移动应用程序和电子应用程序。如果你愿意，都使用相同的代码库！
