::: tip
你可能还想看看我们自己的HTML 5视频播放器组件。[QMediaPlayer](https://github.com/quasarframework/app-extension-qmediaplayer)，它比QVideo(基本上是一个指向嵌入Youtube视频的iframe)高级得多。
:::
