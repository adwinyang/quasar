QPagination组件可用于任何需要分页系统的地方。它为用户提供了一个在项目或页面之间移动的简单用户界面。

QPagination有两种操作模式：只使用按钮或使用输入框。后者允许用户通过点击/敲击输入框，输入页码，然后按回车键，转到一个特定的页面。如果新的页码在有效范围内，模型将被相应改变。
