QSkeleton是一个组件，用于在加载实际页面数据之前显示内容的占位符预览。这是一种很好的方式，可以在页面完全加载之前告知用户对它的期望，并提高感知性能。它可以被用来在获取数据时在屏幕上逐步显示信息。
