QAvatar组件创建了一个可缩放的、可着色的元素，在其形状中可以有文本、图标或图像。默认情况下，它是圆形的，但它也可以是方形的，或者应用一个边框半径，使方形的边角变圆。

它经常与其他组件的插槽一起使用。
