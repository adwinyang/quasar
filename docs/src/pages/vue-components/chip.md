QChip组件基本上是一个简单的UI块实体，例如以一种紧凑的方式表示更高级的底层数据，如联系人。

芯片可以包含实体，如头像、文本或图标，也可以选择有一个指针。如果经过配置，它们也可以被关闭或删除。

::: tip
还可以查看[QBadge](/vue-components/badge)。
:::
